HTML

Ideas principales

  • La mayoria de las etiquetas deben tener un inicio y un fin.
  • El inicio de las etiquetas es normal, es decir que lleva los dos caracteres y dentro de estos se encuentra el nombre.
  • Para el cierre de la etiqueta antes de escribir el nombre de la etiqueta debemos escribir una diagonal.
  • Entre el inicio y el fin de la etiqueta debe estar el contenido.

Apuntes

HTML es un lenguaje de marcado utilizado para la creacion de paginas web. En este lenguaje de marcado se utilizan comandos llamados etiquetas, las cuales nos sirven para distintos usos como lo son colocar secciones, crear enlaces, colocar texto e imagenes, entre muchas otras cosas. Las etiquetas estan estructuradas por un inicio y final en algunos casos, estas deben ser escritas entre dos caracteres, de lo contrario, no funcionaran.

Resumen

HTML es un lenguaje que utiliza las etiquetas para generar distintos elementos, la manera de usarse es muy sencilla y practica para aquellos que lo utilizan.

Texto en HTML

  • Se conforma por encabezados, parrafos y listas los cuales nos ayudan a colocar el contenido de una manera estructurada.
  • Una de las principales funciones de HTML es dar al texto estructura y significado de forma que un navegador pueda mostrarlo correctamente.
  • Existen tanto listas ordenadas (las cuales enumeran los elementos), como las listas desordenadas (las cuales solamente muestran un punto al lado del elemento).

En HTML tenemos distintas maneras de colocar texto. De primera mano tenemos los encabezados los cuales son escritos como "<.h1>" hasta el "<.h6>", conforme el numero es mas grande este se vuelve un subtitulo del anterior. Es recomendable solamente usar un "<.h1>" por pagina.

En HTML, cada parrafo tiene que estar delineado por un elmento"<.p> <. /p>".

En HTML existen dos tipos de listas, la desordenada muestra a los elementos marcados en la parte izquierda por un simbolo, las ordenadas muestra los elementos de manera enumerada.

HTML nos ofrece una manera sencilla de estructurar el texto a partir de dividir el contenido en distintas etiquetas que a la vez tienen sus propias variantes.

Imagenes en HTML

Ideas principales

  • Para colocar imagenes debemos contar con una carpeta hermana de nuestro index, ambos deben estar en el mismo folder de proyecto. Tambien podemos colocar imagenes en liena utilizando el url de la imagen.
  • El atributo "src" se utiliza para establecer la ruta por la cual se colocara la imagen.
  • El atributo "width" determina el ancho de la imagen.
  • El atributo "alt" determina el alto de la imagen.

Apuntes

La imagenes en la web no solo sirven para cuestiones esteticas, las imagenes cuentan historias, ayudan a los buscadores a indexar las paginas, brindan a los usuarios confinza y buena experiencia de usuario.

Las imagenes proporcionan un descanso en la lectura y muchas veces sirven de explicacion grafica para lo que se esta contando.

Las imagenes pueden ser ajustadas con los atributos "width" y ""alt.

El atributo "src" es desde donde asignaremos la imagen al index, podemos colocar una ruta desde una carpeta especial para imagenes o colocar la url de una imagen en linea.

Resumen

En HTML podemos colocar imagenes y alterar sus dimensiones segun se requiera, esto nos ayuda a moldear la estetica y mejorar la experiencia del usuarion

En HTML podemos colocar links de distintas manera, ya sea como titulos, imagenes o simple texto. Esto nos ayuda a generar una pagina mas dinamica a partir de la modificacion de los atributos como target o title.
